# 6.1 标准签单步骤

<h3>6.1 标准签单步骤（缺一不可）</h3><p>本环节为香港保险签约核心落地环节，所有步骤必须严格按顺序执行、缺一不可，全程由香港持牌顾问主导，严格遵循香港保监局监管要求，做到全程合规、全程留痕；转介人严格坚守角色边界，仅做好现场衔接提醒，不介入任何专业操作及销售类动作。</p><h4>步骤1：客户本人持证件过关（核心前提，合规底线）</h4><ul><li>证件要求：有效身份证原件、有效港澳通行证（含有效签注，剩余有效期不少于3个月）。</li><li>过关流程：客户自行持证件通过香港正规口岸过关（如罗湖口岸、福田口岸、香港国际机场等）。</li><li>持牌顾问动作：提前1天再次提醒客户核对证件；过关后在约定地点接应客户，现场核对身份及证件有效性。</li><li>注意事项：若证件存在过期、信息不符等问题，立即终止当日签单流程，协助客户调整证件后重新预约。</li></ul><h4>步骤2：前往正规门店（合规签约场所，杜绝违规）</h4><ul><li>门店要求：必须是香港保险公司官方授权的线下门店，具备香港保监局认可的签约资质，有录音录像设备。</li><li>前往流程：持牌顾问带领客户从口岸或住宿地点，前往提前预约好的正规门店。</li><li>转介人注意事项：可陪同客户前往门店，但仅在等候区等候，不进入签约区域，不参与任何签约相关操作。</li></ul><h4>步骤3：合规讲解风险披露（监管强制要求，全程录签）</h4><ul><li>讲解核心内容：香港保险与内地保险的监管差异、保障范围差异、理赔流程差异；产品核心风险（分红不确定性、汇率波动风险、提前退保损失、免责条款）；签约及后续服务规则；合规提示。</li><li>客户确认：讲解完毕后，客户明确回复"理解"后，签署《风险披露声明书》，严禁代签。</li><li>合规留痕：录音录像文件、《风险披露声明书》同步归档，永久留存可查。</li></ul><h4>步骤4：填写健康告知问卷/投保书（如实填写，避免拒赔）</h4><ul><li>填写引导：持牌顾问逐题引导客户阅读，用通俗语言讲解，告知客户如实填写过往病史、体检异常、用药情况等。</li><li>核对确认：填写完毕后，持牌顾问逐页核对，确认填写内容完整、无涂改，在所有需签字处亲笔签署姓名、日期，严禁代签、漏签。</li><li>合规留痕：《健康告知问卷》《投保书》归档，同步留存客户签署过程的录像记录。</li></ul><h4>步骤5：现场缴费（合规缴费，确保保单启动）</h4><ul><li>缴费方式：香港本地银行卡（最便捷合规）、内地银联卡（部分支持）、现金缴费（需符合相关规定）。</li><li>注意事项：严禁第三方代缴、代转保费，缴费人必须为客户本人；若缴费失败，持牌顾问协助客户调整缴费方式，确保当日完成首期保费缴纳。</li><li>合规留痕：《缴费收据》客户留存一联，另一联归档，同步留存缴费过程记录。</li></ul><h4>步骤6：提交核保（收尾环节，启动保单审核）</h4><ul><li>资料整理：持牌顾问核对所有签约资料（《投保书》《健康告知问卷》《风险披露声明书》、证件复印件、缴费收据等）。</li><li>提交核保：当场将资料提交至核保部门，在系统录入投保信息，生成投保编号，告知客户核保周期（通常3-7个工作日）。</li><li>后续衔接：向客户发放《签约回执》，告知后续保单生效、保单送达等相关事宜。</li></ul><h4>核心纪律要求</h4><ul><li>所有步骤必须按顺序执行，缺一不可，严禁跳过、简化任何环节。</li><li>全程由香港持牌顾问主导，所有操作全程录音录像、全程留痕。</li><li>客户必须本人亲自参与所有环节，严禁代签、代办、代缴。</li><li>转介人不介入任何签单专业操作，仅做好衔接提醒，不越位、不缺位。</li><li>严禁任何违规操作（如隐瞒风险、诱导填写、违规缴费），确保保单合法有效。</li></ul>
