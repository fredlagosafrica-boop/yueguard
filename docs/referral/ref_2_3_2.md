# 3.2 筛选目的

<h3>3.2 筛选目的</h3><h4>一、核心筛选定位</h4><p>通过前期标准化4项精细化问诊+规则前置告知，做精准客户分层筛选，提前剔除无效、低质量意向，把时间和精力聚焦服务有真实投保刚需、匹配港险准入条件的精准客户。</p><h4>二、四大劣质客户精准过滤维度</h4><p><strong>只想随便问问、无实际落地意愿：</strong>仅好奇了解、随口咨询，没有实际投保规划，不做长期配置打算，只比价、不决策，浪费沟通时间。</p><p><strong>不愿赴港签约、想代办代签：</strong>明知港险合规要求必须本人赴港面签，仍坚持想异地代办、线上签、找人代录，不符合监管规则，无法合规操作。</p><p><strong>过度追求高返佣、只看利益不看保障：</strong>核心关注点不在产品规划、家庭保障和资产配置，一味索要高额返佣，价值观不合，后续容易纠纷、退保、口碑隐患大。</p><p><strong>预算严重过低、不符合港险起投门槛：</strong>年缴预算远低于香港保险基础起投标准，达不到产品准入门槛，无法做合理方案配置，无实际成交可行性。</p><h4>三、筛选核心价值</h4><p>节省时间成本，不消耗精力在无效闲聊和低意向客户身上；提前规避不合规诉求，减少后续沟通矛盾与纠纷；精准锁定有预算、有需求、能赴港、理念同频的优质客户，提升成交率与服务效率；打造高质量客户圈层，专注做长期合规保单服务与后续维护。</p>
