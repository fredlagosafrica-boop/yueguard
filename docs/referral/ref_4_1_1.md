# 4.1.1 合规合作资源搭建

<h3>4.1.1 合规合作资源搭建</h3><p>在香港保险转介合作中，合规是生命线。自2025年10月1日起，香港保监局明确转介费上限为首年佣金的50%，且合作方必须为持牌实体。以下从资源筛选、协议签订、流程管控三方面，搭建完整的合规合作体系。</p><h4>一、合作资源筛选（唯一入口：香港保监局持牌登记册）</h4><h5>1. 核验渠道（必备）</h5><p>官网查询：香港保监局持牌中介登记册 <a href="https://iir.ia.org.hk" target="_blank">https://iir.ia.org.hk</a></p><p>核验信息：公司全称、牌照号码（FB/GB开头）、有效期、业务范围（长期／一般）</p><h5>2. 主流合规经纪公司参考（持牌+内地服务成熟）</h5><table><thead><tr><th>公司名称</th><th>牌照号</th><th>核心优势</th></tr></thead><tbody><tr><td>康宏理财服务有限公司</td><td>FB1360</td><td>大湾区服务成熟，内地客户案例多</td></tr><tr><td>京东保险顾问（香港）有限公司</td><td>FBxxxx</td><td>科技化流程，线上对接便捷</td></tr><tr><td>赛诺保险经纪（大湾区）有限公司</td><td>FB1280</td><td>深耕内地高净值客户</td></tr></tbody></table><h5>3. 绝对禁止合作的对象</h5><ul><li>❌ 无香港保监局牌照的个人／公司</li><li>❌ 牌照过期／被吊销的机构</li><li>❌ 声称"可私下返佣""突破50%上限"的中介</li></ul><h4>二、转介协议核心条款（合规铁律，缺一不可）</h4><h5>1. 主体条款</h5><ul><li>甲方（合作方）：香港保监局持牌经纪公司／顾问（附牌照复印件）</li><li>乙方（转介方）：内地合规企业／个人（非持牌），明确"仅提供客户线索，不参与销售"</li></ul><h5>2. 费用条款（监管红线）</h5><ul><li>转介费比例：≤首年佣金（经纪公司实收）的50%（2025.10.1起强制执行）</li><li>支付条件：客户赴港签约+过21天冷静期+保费到账后支付</li><li>禁止条款：无返佣、无现金支付、无私下分成，所有费用走公账并开票</li></ul><h5>3. 角色合规条款（保监局核心要求）</h5><p>乙方（转介方）不得：提供保险产品建议、讲解条款、促成签约、代收保费</p><p>甲方（持牌方）必须：所有受规管活动（产品讲解、风险披露、签约）在香港境内由持牌人完成</p><h4>三、全流程合规管控（从对接至售后）</h4><h5>1. 前期对接</h5><ul><li>核验牌照原件→留存官网查询截图→签订NDA（保密协议）</li><li>明确双方对接人：甲方（持牌顾问）、乙方（转介专员），禁止私下对接</li></ul><h5>2. 客户转介流程（标准SOP）</h5><ul><li>乙方：收集客户基本信息（姓名、联系方式、投保意向）→仅推送线索，不做产品推荐</li><li>甲方：持牌顾问1对1联系客户→香港境内完成需求分析、产品匹配、风险披露</li><li>签约环节：客户亲身赴港，在甲方合法营业场所签署保单→保费直接支付给保险公司，不经过乙方账户</li><li>售后：甲方负责保单管理、理赔服务；乙方协助内地客户沟通，不参与任何受规管活动</li></ul><h5>3. 合规留痕（监管检查必备）</h5><ul><li>转介协议原件+牌照复印件+费用支付凭证（公账流水、发票）</li><li>客户沟通记录：乙方仅推送线索的聊天截图；甲方在港服务的全程录音／录像（合规场所）</li><li>定期自查：每季度核对转介费比例，确保未超过50%上限</li></ul><h4>四、常见违规风险警示</h4><ul><li>❌ 超比例转介费：超过50%需向保监局特殊说明，无合规理由将被处罚</li><li>❌ 内地签约／收款：属于"地下保单"，不受香港法律保护，保单可能作废</li><li>❌ 返佣行为：直接违反保监局规定，双方均面临罚款+牌照吊销风险</li></ul>
