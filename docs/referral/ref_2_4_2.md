# 4.2 角色边界

<h3>4.2 角色边界</h3><h4>一、核心原则</h4><p>三方建群完成、完成人员介绍与信任背书后，转介绍人立刻退出所有销售类环节，严格守住角色边界，不越位、不抢位、不替代专业持牌顾问做专业销售动作。</p><h4>二、转介绍人严禁参与的事项</h4><ul><li>不参与任何产品讲解、险种对比、条款解读；</li><li>不参与方案设计、保费测算、收益演示、计划书解读；</li><li>不做利益承诺、分红预判、理赔承诺、政策解读；</li><li>不私自给客户推荐产品、不擅自修改规划方案、不解答专业合规及产品细节问题；</li><li>不参与议价、返佣沟通、费率私下承诺等敏感销售行为。</li></ul><h4>三、转介绍人保留的合法角色定位</h4><p>只保留居中衔接、氛围维护、流程提醒、规则同步、矛盾协调的服务角色：</p><ul><li>负责前期已讲过的基础规则重申（赴港面签要求、基础流程说明）；</li><li>负责维系群内沟通氛围，提醒客户及时回复、配合对接；</li><li>若客户有情绪顾虑、沟通卡点，居中做协调传话；</li><li>不碰专业产品、不做专业销售输出，把专业事完全交给香港持牌顾问。</li></ul><h4>四、设置角色边界的三大目的</h4><ul><li><strong>合规风控：</strong>转介绍人无香港持牌资质，不做无证销售讲解，规避监管合规风险；</li><li><strong>专业分工：</strong>专业方案、产品解读由持牌顾问负责，更权威、更专业、更严谨；</li><li><strong>责任切割：</strong>避免后续收益、条款、理赔等纠纷牵连转介绍人，权责清晰、各司其职；</li><li><strong>提升成交效率：</strong>减少多头沟通、信息混乱，客户只对接一位专业顾问，决策更顺畅。</li></ul><h4>五、可直接套用群内话术</h4><p>我这边前期已经帮您把需求梳理完毕，接下来产品细节、方案定制、收益解读、条款疑问都由香港持牌顾问为您专业一对一讲解。我只负责全程衔接协调，不参与专业产品解读和方案设计，确保给到您的都是香港正规持牌顾问的合规专业建议，请您放心对接。</p>
