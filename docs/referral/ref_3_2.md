# 二、明确禁止违规行为（零容忍）

<h3>二、明确禁止违规行为（零容忍）</h3><h4>严禁内地远程签单</h4><p>禁止在内地任何城市通过线上沟通、邮寄保单、异地寄送资料等方式完成投保签单，不接受内地异地落地签单操作。</p><h4>严禁视频隔空签单</h4><p>禁止以微信、腾讯会议、视频通话、直播连线等任何线上视频形式，进行投保讲解、身份核验、保单签署及见证流程。</p><h4>严禁代签代署</h4><p>任何情况下不得代替客户签署投保单、健康告知书、风险披露文件、权益确认书等所有合规文件，禁止任何人冒用客户笔迹、代为签名。</p><h4>严禁代填投保资料</h4><p>禁止业务员、中间人、转介人私自替客户填写个人信息、健康告知、财务状况、职业信息、投保意愿等投保核心资料；所有信息必须由客户本人如实核对、亲笔确认。</p><h4>严禁代办全套流程</h4><p>不得绕过客户本人，私下代办投保申请、风险测评、见证核验、回执签收等任一环节，全程必须客户本人亲自参与。</p>
