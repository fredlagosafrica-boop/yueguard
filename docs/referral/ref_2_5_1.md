# 5.1 持牌顾问核心合规动作

<h3>5.1 持牌顾问核心合规动作（全程可追溯、全流程留痕）</h3><p>本环节为香港保险签约全流程唯一专业输出、合规风控核心节点，由香港保监局持牌保险顾问独立、完整完成所有动作，转介绍人全程不介入、不干预、不替代，所有操作严格符合香港保险监管条例，每一步均留存书面/聊天/录音记录，实现全程可追溯、权责可界定，从源头规避违规风险、售后纠纷。</p><h4>动作一：闭环需求复核与精准分析</h4><p><strong>执行标准：</strong></p><ul><li>完整同步、复核转介绍人前置收集的4项核心客户信息，不向客户重复提问，用通俗语言完整复述客户需求，获得客户明确认可。</li><li>结合客户身份、预算区间、缴费年限、赴港时间规划，深度拆解核心诉求，区分主次需求。</li><li>同步前置合规提醒：明确告知客户香港保险与内地保险的监管差异、保障范围、免责规则、汇率风险、流动性规则。</li></ul><p><strong>合规留痕要求：</strong>留存需求确认文字记录、客户需求拆解和风险告知的文字/语音记录，归档至客户服务档案。</p><p><strong>核心目的：</strong>完全对齐客户真实诉求，从源头降低后续退保、纠纷风险，实现服务全程可追溯。</p><h4>动作二：量身定制专属方案</h4><p><strong>执行标准：</strong></p><ul><li>严格以客户预算上限、缴费年限、核心需求为唯一依据，定制1-2套最优适配方案，绝不推荐超出预算、偏离核心诉求的产品。</li><li>方案内容完整规范：明确标注产品名称、保障责任、缴费周期、年度保费、保障期限、预期收益、现金价值规则、免责条款、理赔规则、退保损失等核心信息。</li><li>方案对比通俗易懂，用客户能听懂的大白话讲解方案差异，明确告知每套方案的优势、适配场景、潜在风险。</li><li>严守合规红线：不承诺保证收益、不预判分红实现率、不做刚性兑付承诺，所有演示数据严格按照保险公司官方演示标准标注。</li></ul><p><strong>合规留痕要求：</strong>定制方案计划书完整存档，发送给客户的方案文件、文字解读记录全程留存。</p><p><strong>核心目的：</strong>实现一人一方案、完全贴合客户情况，守住合规底线，杜绝信息差导致的后续纠纷。</p><h4>动作三：完整条款讲解与风险充分告知</h4><p><strong>执行标准：</strong></p><ul><li>核心保障责任讲解：明确保障范围、赔付标准、理赔条件、理赔流程，不夸大保障责任。</li><li>免责条款逐条明确：重点讲解合同免责条款、除外责任，确保客户完全知情。</li><li>关键规则充分告知：保单流动性限制、提前退保损失、分红实现率不确定性、汇率波动风险、缴费规则、保单保全规则。</li><li>客户疑问全解答：针对客户所有问题给予真实、准确、合规的答复，不懂的内容绝不随意答复。</li></ul><p><strong>合规留痕要求：</strong>条款讲解、风险告知的文字/语音记录完整留存；客户所有疑问及对应解答记录完整归档。</p><p><strong>核心目的：</strong>完全履行监管要求的告知义务，确保客户知情同意、明白签约，彻底杜绝投诉纠纷。</p><h4>动作四：全流程合规留痕与知情确认</h4><p><strong>执行标准：</strong></p><ul><li>严格按照香港保监局监管要求，完成客户投保前知情确认、风险提示确认、需求匹配确认。</li><li>全程固定合规证据：从需求对接、方案讲解、条款告知到赴港确认，所有沟通环节完整留存。</li><li>签约前最终合规确认：再次明确告知客户香港保险必须本人亲临香港线下网点、现场录音录像面签，严禁异地代办。</li><li>严禁任何违规动作：绝不参与返佣、利益输送、暗箱操作。</li></ul><p><strong>合规留痕要求：</strong>客户投保前知情确认、风险确认、签约规则确认的文字记录完整存档；客户全周期服务档案闭环管理。</p><p><strong>核心目的：</strong>完全符合香港保险监管合规要求，实现全流程闭环可追溯，隔离合规风险。</p><h4>动作五：赴港签约前置筹备与全流程指引</h4><p><strong>执行标准：</strong></p><ul><li>结合客户赴港时间规划，一对一预约保险公司官方签约网点，提前发送完整赴港准备清单。</li><li>提前同步完整签约流程：告知客户赴港当天的签约动线、办理时长、开户流程、缴费流程、保单生效规则。</li><li>赴港前最终复核：再次复核客户方案、保费、保障内容、签约规则，确认客户无任何疑问。</li><li>全程专属对接：赴港前、赴港途中、签约现场全程一对一响应，现场全程陪同办理。</li></ul><p><strong>合规留痕要求：</strong>赴港预约记录、资料清单、流程指引发送记录完整留存；签约前最终确认记录、现场陪同办理的合规记录完整归档。</p><p><strong>核心目的：</strong>提前扫清所有签约障碍，确保客户顺利完成合规面签，为后续长期售后维护奠定基础。</p><h4>本环节刚性纪律要求</h4><ul><li>所有专业动作仅限香港正规持牌保险顾问完成，无牌人员严禁参与任何产品讲解、方案设计、条款解读相关动作。</li><li>全程严禁夸大宣传、误导销售、隐瞒条款、承诺收益、违规返佣，所有操作严守香港保监局监管规定。</li><li>转介绍人仅可做氛围维护、流程衔接、居中协调，严禁介入任何专业销售、合规告知相关环节。</li></ul>
