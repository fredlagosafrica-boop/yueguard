# 8.1 转介费结算触发条件

<h3>8.1 转介费结算触发条件</h3><h4>一、核心触发条件（三项需同时满足，无任何例外）</h4><ul><li><strong>1. 保单已过21天法定冷静期：</strong>冷静期自保单正式出具、客户签署确认之日起计算，满21个自然日。冷静期计算以香港保险公司系统记录的保单出具日期为准，截止日期精确到自然日，需确认冷静期已完整届满。</li><li><strong>2. 客户未办理任何退保手续：</strong>冷静期内及冷静期届满后，直至转介费结算启动前，客户未提出任何书面或口头退保申请，无任何退保记录。验证以香港保险公司系统查询结果为准，保单状态需为"有效存续"。</li><li><strong>3. 保单已正常承保且正式生效：</strong>核保结果为"正常承保"（不含加费承保、除外责任承保），客户已按投保约定足额缴纳首期保费，无保费拖欠、漏缴情况；保单已完成所有生效流程，保险公司系统显示"正式生效"状态。</li></ul><h4>二、补充说明</h4><ul><li>若核保结果为加费承保、除外责任承保，即便满足"过冷静期、未退保"条件，也不触发转介费结算，仅"正常承保"且生效的保单可参与结算；</li><li>冷静期内客户若提出退保申请，无论最终是否撤销，均视为未满足结算条件，不予结算；</li><li>保单生效后，若在转介费结算前发生退保，立即终止结算流程，已启动的结算需暂停并撤销；</li><li>转介费结算的所有条件验证，均以香港保险公司系统记录、正式出具的保单文件及相关凭证为准，无任何口头约定、例外变通。</li></ul>
