# 6.3 绝对红线

<h3>6.3 绝对红线（严禁触碰）</h3><p>所列红线为香港保险签约全流程不可逾越的合规底线，转介人、香港持牌顾问均需严格恪守，零例外、零容忍。一旦违反，将直接导致保单无效、触发香港保监局监管处罚，相关责任人需承担相应法律责任。</p><h4>一、绝对禁止操作（每条均为硬性红线，无任何变通空间）</h4><ul><li><strong>严禁内地远程录单：</strong>严禁在中国大陆境内，通过任何远程方式办理香港保险录单相关操作，包括远程录入客户投保信息、远程采集影像资料、远程提交录单申请等。所有录单操作必须在香港保险公司正规线下门店内完成，且客户本人必须在场。</li><li><strong>严禁视频签单：</strong>严禁以视频连线、线上会议、语音通话等任何远程形式，开展香港保险签约相关操作，包括视频讲解风险披露、视频引导客户签署文件等。无论何种特殊情况，均不允许通过视频方式替代客户本人亲临香港线下签约。</li><li><strong>严禁代签投保书：</strong>严禁任何人员替代客户本人签署《投保书》《风险披露声明书》等任何与投保相关的文件。所有需客户签署的文件，必须由客户本人在香港保险公司正规门店内，亲笔签署，严禁代签、冒签、补签。</li><li><strong>严禁代填健康告知：</strong>严禁任何人员替代客户填写《健康告知问卷》及其他与客户健康相关的投保资料。健康告知需由客户本人亲自、如实填写，持牌顾问仅可提供题目含义的通俗讲解，严禁代填、诱导填写、隐瞒填写。</li><li><strong>严禁客户在内地签署任何投保相关文件：</strong>所有与香港保险投保相关的文件（包括投保书、健康告知问卷、风险披露声明书、缴费确认单等），均不允许客户在中国大陆境内签署。</li></ul><h4>二、红线界定补充（明确边界，杜绝模糊地带）</h4><ul><li><strong>远程录单界定：</strong>无论设备所在地、操作人所在地，只要录单操作发生在内地，或通过内地网络、内地设备完成，均属于违规远程录单。</li><li><strong>代签/代填界定：</strong>无论是否经过客户口头同意、书面授权，任何替代客户签署文件、填写投保资料的行为，均属于违规。</li><li><strong>内地签署界定：</strong>客户在中国大陆境内签署任何投保相关文件，均属于违规，即使文件后续带回香港归档，仍视为无效签署。</li></ul><h4>三、违规后果警示</h4><ul><li><strong>保单层面：</strong>违规操作将直接导致保单无效，客户已缴纳的保费可能无法全额退还。</li><li><strong>监管层面：</strong>香港持牌顾问违规将被香港保监局吊销牌照、处以高额罚款；转介人违规将被纳入行业黑名单。</li><li><strong>法律层面：</strong>违规行为涉嫌违反香港保险监管条例及相关法律法规，相关责任人需承担相应法律责任。</li><li><strong>责任层面：</strong>因违规操作给客户、保险公司造成的一切损失，均由违规方全额承担。</li></ul><h4>四、执行要求</h4><p>转介人、香港持牌顾问需全程牢记上述红线，严格执行合规流程，不抱任何侥幸心理、不搞任何变通操作。若发现违规苗头，需立即终止相关操作，及时纠正。</p>
