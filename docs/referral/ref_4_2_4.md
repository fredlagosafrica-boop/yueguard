# 4.2.4：微信号/企微标签体系（精准触达）

<h3>4.2.4 微信号/企微标签体系（精准触达）</h3><p>私域引流后，第一时间打标签是精准运营的基础。标签体系决定后续朋友圈内容推送、客户分层运营、一对一跟进的质量，直接影响转化率。</p><h4>一、标签体系设计（核心5类，直接可用）</h4><table><thead><tr><th>标签</th><th>含义</th><th>运营动作</th></tr></thead><tbody><tr><td>意向-教育金</td><td>关注孩子教育规划</td><td>推送教育金配置文章、案例</td></tr><tr><td>意向-养老金</td><td>关注退休规划</td><td>推送养老规划攻略、产品对比</td></tr><tr><td>意向-保障型</td><td>关注重疾/医疗保障</td><td>推送保障型产品分析、理赔案例</td></tr><tr><td>已成交</td><td>已签单客户</td><td>推送续费提醒、保单服务</td></tr><tr><td>高潜力</td><td>高净值/企业主</td><td>一对一深度跟进，定制方案</td></tr></tbody></table><h4>二、标签打标时机与话术</h4><h5>1. 加好友时自动打标（基础标签）</h5><p>系统自动打上「新增客户」标签，确保新客户不遗漏。</p><h5>2. 首次沟通后打标（需求确认）</h5><p>根据首次沟通内容，判断客户需求方向，手动打上对应意向标签。</p><p><strong>话术示例：</strong></p><ul><li>客户主动问教育金 → 打「意向-教育金」</li><li>客户问养老产品 → 打「意向-养老金」</li><li>客户关心重疾保障 → 打「意向-保障型」</li><li>客户提及资产量级/企业主 → 打「高潜力」</li></ul><h5>3. 签单后打标（客户分层）</h5><p>签单完成后，更新标签为「已成交」，同时保留意向标签，便于后续精准推送续费和服务信息。</p><h4>三、朋友圈精准触达策略</h4><h5>按标签分组推送（企微/微信号均适用）</h5><ul><li><strong>意向-教育金：</strong>每周1条教育金相关，专业+案例结合</li><li><strong>意向-养老金：</strong>每周1条养老规划相关，政策解读+产品分析</li><li><strong>意向-保障型：</strong>每周1条保障型内容，理赔案例+产品对比</li><li><strong>已成交：</strong>每月1条保单服务提醒（续费/周年/理赔）</li><li><strong>高潜力：</strong>一对一深度维护，定制化内容推送</li></ul><h4>四、标签管理注意事项</h4><ul><li>标签不超过5类，优先标注高意向客户</li><li>每月定期清理标签，移除长期无互动的客户（降级为「沉默客户」）</li><li>标签仅自己可见，不展示给客户</li><li>企业微信标签可与CRM系统打通，实现自动打标</li></ul>
