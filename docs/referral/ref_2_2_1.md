# 2.1 标准咨询话术

<h3>2.1 标准咨询话术 细分完整版</h3><h4>一、开场礼貌铺垫（温柔亲和版）</h4><p>您好～ 先跟您说明一下我的服务范围哈：</p><p>我这边只专注做香港保险转介绍资源对接，不单独做产品细节讲解、不做个人理财投资建议、也不做私下方案承诺。</p><p>我的核心工作，是根据你的真实需求，帮你精准匹配香港正规持牌顾问，全程对接合规专业、一对一专属咨询服务。</p><h4>二、身份与合规提前告知（建立信任+规避合规风险）</h4><p>所有对接的顾问都是香港保险业正规持牌代理人/经纪，受香港保监局监管，流程合规、签约全程正规线下见证，保障你的资金安全和权益合规。</p><p>我先简单帮你做个基础需求摸底，方便给你匹配最合适、擅长你需求类型的持牌顾问，不浪费你时间。</p><h4>三、分层精准信息问询（逐条细化，可逐句使用）</h4><p><strong>1. 个人基础信息问询</strong></p><p>方便先说下你的大概年龄吗？不同年龄适配的港险产品类型、费率和配置逻辑差别挺大的。</p><p><strong>2. 年度预算&amp;缴费规划问询</strong></p><p>想了解下你每年能稳定拿出多少预算配置港险？另外计划准备缴费多少年？常见的有5年、10年、15年、20年或趸交，我好帮你对应匹配产品结构。</p><p><strong>3. 核心配置目的细分问询（逐条引导，给客户选项）</strong></p><p>你配置香港保险，主要偏向哪一种核心需求呀？</p><ul><li>一是长期储蓄增值、稳健理财，做资产保值、美金资产配置；</li><li>二是重疾、医疗、人寿保障，补充内地医保和重疾缺口；</li><li>三是孩子教育金、成年婚嫁金，提前锁定未来孩子求学、成家刚性开支；</li><li>四是资产隔离、财富传承、税务规划，做家庭资产保全、代际传承。</li></ul><p>你可以跟我说主要侧重哪一项，有兼顾的也可以一起说。</p><h4>四、收尾过渡话术（承接后续对接）</h4><p>好的，我记下你的年龄、年度预算、缴费年限和核心需求后，马上帮你匹配对应擅长该板块的香港持牌顾问。后续由持牌顾问一对一跟你详细讲解产品、方案规划、收益演示、投保流程和合规须知，全程专业答疑，我这边只做对接引荐哈。</p><h4>五、简约干练版（适合微信快速发消息）</h4><p>您好～ 我只负责香港保险转介绍对接，不做产品讲解、不提供投资建议。帮你匹配香港正规持牌顾问，需要简单了解3个信息：①你的年龄；②每年可投入港险的预算；③计划缴费几年；④配置目的：储蓄增值/重疾医疗/孩子教育金/资产传承隔离，告诉我你的需求即可，我帮你精准匹配专属顾问。</p>
