# 7.1 核保流程及结果

<h3>7.1 核保流程及结果</h3><p>核保是香港保险签约后、保单生效前的核心环节，由香港保险公司专业核保部门独立开展，全程遵循香港保监局监管要求，确保审核严谨、流程合规、结果公正。</p><h4>一、核保周期</h4><ul><li><strong>常规核保周期：</strong>通常为3-7个工作日，适用于客户健康状况良好、投保资料完整无误、无缺失的普通案例。</li><li><strong>延长核保周期：</strong>复杂案例的核保周期可能延长至1-2周，主要适用于：健康告知存在异常、投保高保额保单、客户身份特殊（如企业主需补充资产证明）、投保特殊险种（重疾险、高端医疗险等）。</li></ul><h4>二、核保结果（5类）</h4><ul><li><strong>正常承保：</strong>客户健康状况符合投保要求，按约定保额、保费、缴费年限正常承保，无任何附加条件。后续：第一时间同步客户，发送电子保单初稿供核对，确认后正式出具保单，进入21天法定冷静期。</li><li><strong>要求体检：</strong>健康告知存在轻微异常，需补充指定体检项目报告。后续：持牌顾问明确体检要求，协助预约体检；体检报告提交后3-5个工作日内完成二次审核。</li><li><strong>除外责任承保：</strong>存在明确健康异常，核保部门同意承保但排除该异常及相关并发症的赔付责任。后续：持牌顾问详细解读除外责任范围，客户签署除外责任确认书后出具保单。</li><li><strong>加费承保：</strong>健康状况存在一定风险但未达拒保标准，同意承保但需在原保费基础上增加一定比例保费。后续：向客户明确加费金额及原因，客户签署加费确认书后出具保单。</li><li><strong>拒保：</strong>健康状况或个人情况不符合产品投保要求。后续：如实告知拒保结果，协助办理已缴保费退还手续，告知后续可选择的其他合规规划方向。</li></ul><h4>三、核保相关补充要求</h4><ul><li>核保结果具有唯一性、权威性，转介人、持牌顾问均无权干预、修改核保结果。</li><li>客户需积极配合核保部门的补充要求，若拒绝配合，核保部门将按拒保处理。</li><li>持牌顾问需每3个工作日同步一次核保进度，转介人可向客户传递进度信息，但不解读核保细节、不承诺核保结果。</li><li>所有核保资料、核保结果、沟通记录均需全程归档，确保合规可追溯。</li></ul>
