# 6.2 转介人可做动作

<h3>6.2 转介人可做动作</h3><p>转介人在签单环节可选择陪同客户赴港，核心角色定位为「陪同办事、协助沟通」，严格坚守4.2章节角色边界，全程不介入任何产品讲解、条款解读、方案设计、风险披露等销售类、专业类动作，仅提供基础陪同与沟通衔接服务。</p><h4>一、可选陪同赴港（自愿选择，不强制、不绑定）</h4><p>转介人可根据自身时间安排、客户需求，自主选择是否陪同客户赴港办理签单事宜，陪同与否不影响签单流程推进；若选择陪同，需全程遵守角色边界，聚焦辅助服务。</p><h4>二、核心可做动作</h4><p><strong>（一）陪同办事：提供基础陪同服务，缓解客户顾虑</strong></p><ul><li><strong>行程陪同：</strong>陪同客户从内地出发、过关、前往香港保险公司正规门店，全程陪同客户往返，缓解客户独自赴港的陌生感。</li><li><strong>现场等候：</strong>抵达保险公司门店后，仅在门店等候区等候客户，不进入签约区域，不围观、不干扰持牌顾问与客户的签单操作。</li><li><strong>基础协助：</strong>协助客户处理签单现场的非专业事务（如提醒客户携带证件、咨询卫生间位置等），不触碰任何签单相关的专业资料和操作。</li><li><strong>返程陪同：</strong>签单完成后，可陪同客户返程，协助处理返程中的基础事宜，全程不讨论签单细节、产品条款等专业内容。</li></ul><p><strong>（二）协助沟通：居中衔接，化解沟通卡点</strong></p><ul><li><strong>信息传递：</strong>若客户与持牌顾问沟通中出现不畅，可居中传递基础信息，协助双方准确传达需求，不解读信息、不干预沟通内容。</li><li><strong>疑问衔接：</strong>若客户有非专业类疑问（如门店位置、返程交通），可协助咨询或结合自身经验解答；若提出产品、条款、收益等专业类疑问，立即引导客户直接对接持牌顾问。</li><li><strong>突发衔接：</strong>若现场出现突发情况（如客户临时身体不适、证件临时找不到），第一时间协助处理，同时同步持牌顾问，居中协调调整安排，不擅自决策。</li></ul><h4>三、严格禁止动作</h4><ul><li>严禁进入签约区域，不围观、不参与任何专业操作；</li><li>严禁向客户讲解任何产品相关内容、解读条款、演示收益、推荐方案；</li><li>严禁干预客户与持牌顾问的沟通，不替客户做决策、不诱导客户签署任何文件；</li><li>严禁参与缴费操作，不替客户代缴保费；</li><li>严禁触碰签单相关的专业资料（投保书、健康告知问卷、风险披露声明书等）；</li><li>严禁向客户做出任何利益承诺、理赔承诺、分红预判。</li></ul><h4>四、核心原则</h4><p>转介人陪同赴港的核心意义是"陪伴与衔接"，核心职责是缓解客户顾虑、协助处理非专业事务、居中传递基础信息，全程不越位、不缺位、不干预专业流程，确保签单流程合规、顺畅。</p>
