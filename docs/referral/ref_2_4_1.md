# 4.1 精细化操作步骤

<h3>4.1 精细化操作步骤</h3><h4>第一步：信息同步</h4><p>把前期问诊收集的4大核心信息完整、无遗漏同步给香港持牌顾问：</p><ul><li>客户预算：年投入保费、缴费年限、总预算区间；</li><li>核心需求：储蓄增值 / 重疾保障 / 教育金 / 养老规划 / 资产隔离 / 税务优化；</li><li>客户身份：内地普通居民 / 企业主 / 自由职业者；</li><li>时间规划：近期是否有赴港计划。</li></ul><p>同步目的：让持牌顾问提前掌握客户情况，不用重复问诊，直接精准定制方案，提升沟通效率。</p><h4>第二步：三方建群</h4><p>建立转介绍人（你）+ 客户 + 香港持牌顾问专属服务小群；</p><p>群内提前明确分工：</p><ul><li>你（转介绍人）：负责前期需求梳理、规则讲解、信任铺垫、居中衔接协调；</li><li>香港持牌顾问：负责专业产品解析、量身定制方案、条款解读、赴港签约指引、后续终身保单售后；</li><li>客户：只需提出疑问、确认需求、配合规划流程即可。</li></ul><h4>第三步：群内背书介绍+角色定位（可直接复制发群里）</h4><p>各位晚上好，给大家做个正式介绍：</p><p>这位是香港正规持牌专属顾问，持有香港保险监管局合法执业牌照，专业负责港险方案定制、条款讲解、赴港面签办理以及保单后续保全、理赔、分红跟进等全流程正规服务，全程合规受香港监管。</p><p>我这边主要负责前期帮您梳理需求、做好规则告知和全程对接协调，后续产品细节、方案配置、专业疑问都可以直接和持牌顾问详细沟通，由顾问为您一对一量身定制最合适的规划方案。</p>
