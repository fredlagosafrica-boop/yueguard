# 9.2 复购与裂变逻辑

<h3>9.2 复购与裂变逻辑</h3><p>复购与裂变的核心根基是"优质合规服务"，转介人需严格恪守角色边界，全程配合香港持牌团队，依托精细化售后维护建立客户信任，推动复购达成、实现裂变闭环。</p><h4>一、复购逻辑：服务留客，需求精准对接</h4><ul><li><strong>筑牢服务基础：</strong>持续落实9.1精细化售后维护要求，做好"桥梁传话"，定期传递价值内容，建立客户信任，为复购创造前提条件。</li><li><strong>精准捕捉需求：</strong>日常互动中温和关注客户需求变化（家庭调整、资产优化、保单升级等），不主动挖掘、不诱导需求，仅被动捕捉客户主动提及的合理需求。</li><li><strong>规范对接流程：</strong>客户明确提出新需求时，第一时间告知"将为您对接香港持牌团队"，快速完整传递需求给持牌顾问，做好衔接工作，不参与任何专业服务环节。</li><li><strong>跟进保障复购：</strong>持续跟进双方沟通进度，确保需求对接无脱节；待复购签约后继续做好售后维护，形成"服务→信任→复购→再服务"良性循环。</li></ul><h4>二、裂变逻辑：口碑裂变，搭建闭环体系</h4><ul><li><strong>夯实裂变基础：</strong>以合规、贴心服务为核心，持续做好售后维护，让客户主动认可服务价值，愿意自发传播、推荐给有需求的亲友。</li><li><strong>合规引导裂变：</strong>不主动提出推荐要求、不设置推荐任务、不以奖励诱导推荐；仅在老客户主动提及"有亲友有需求"时，友好对接，告知可将亲友信息同步给转介人协助对接。</li><li><strong>搭建闭环流程：</strong>老客户推荐新客户后，严格按合规流程为新客户提供服务，同步对接持牌团队；待新客户完成签约后，鼓励新客户进一步自发推荐，形成"老客户推荐→新客户服务→新客户裂变"可持续闭环。</li><li><strong>坚守裂变底线：</strong>严禁以违规形式诱导推荐，严禁虚假宣传；新客户服务流程需严格遵循合规要求，维护口碑。</li></ul><h4>三、复购与裂变核心纪律</h4><ul><li>全程以"优质合规服务"为核心，不依赖违规操作、诱导行为，坚守角色边界。</li><li>尊重客户意愿，复购、裂变均以客户主动意愿为前提，不强迫、不诱导。</li><li>所有复购、裂变相关沟通记录、服务记录均需完整留存，确保可追溯。</li><li>全程配合香港持牌团队，确保客户获得专业、合规的服务。</li></ul>
