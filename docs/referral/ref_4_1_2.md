# 4.1.2 自身人设与能力筹备

<h3>4.1.2 自身人设与能力筹备</h3><p>核心人设：聚焦内地客户核心需求，打造"跨境资产配置顾问+港险合规转介专员"双重标签，以"中立不推销、专业够靠谱、省心全托管"为核心特质，区别于传统中介，成为内地客户对接香港合规港险、规划跨境资产的"信任型桥梁"，既坚守香港保监局合规要求，又贴合内地客户认知习惯，解决客户"怕违规、怕踩坑、怕麻烦、怕被套路"的核心痛点。</p><h4>一、人设内核细化（三大核心特质，落地到具体行为）</h4><h5>（一）中立：不绑定单一机构，只做客户利益优先的筛选者</h5><p>核心逻辑：打破"中介=推销"的刻板印象，明确自身定位为"客户需求匹配者"，而非"某家机构的推广者"，全程以客户实际需求为导向，不夸大产品优势、不隐瞒风险，坚守合规底线。</p><p>具体落地行为：</p><ul><li>不偏袒任何一家香港持牌经纪公司，基于客户需求（如养老、教育、财富传承、保障缺口），筛选2-3家合规持牌机构的适配产品，客观对比优劣势，不强制推荐；</li><li>主动告知客户香港保险与内地保险的差异（如理赔规则、法律适用、货币风险），以及转介流程中的所有合规要求（如转介费比例、签约地点、冷静期规定），不回避潜在风险；</li><li>拒绝"私下返佣""突破50%转介费上限"等违规操作，明确告知客户所有费用透明可查，杜绝任何损害客户利益的暗箱操作。</li></ul><h5>（二）专业：兼具合规知识与跨境资产配置能力，做客户的"懂行伙伴"</h5><p>核心逻辑：专业是人设的核心支撑，既要精通香港保险合规规则，也要懂内地客户的资产配置需求，能够将合规要求与客户需求结合，提供专业、可落地的建议，而非单纯的"线索转介"。</p><p>具体落地行为：</p><ul><li>熟练掌握香港保监局核心规定（如持牌机构核验标准、转介费50%上限、受规管活动范围、冷静期要求），能快速核验合作机构牌照真伪，规避违规合作风险；</li><li>深入了解内地客户常见需求痛点（如跨境资产保值、子女海外教育储备、养老跨境规划、全球医疗保障），结合香港保险产品特点（如美元保单、分红型产品、全球理赔），为客户提供针对性的配置思路；</li><li>能够清晰解答客户常见疑问（如"香港保单在内地是否有效""理赔流程如何操作""转介费由谁承担""赴港签约需要准备哪些材料"），用通俗的语言拆解专业术语，避免客户因信息差踩坑；</li><li>持续更新知识储备，关注香港保监局政策变动、跨境资产配置趋势、香港保险产品迭代，确保给出的建议具备时效性和合规性。</li></ul><h5>（三）省心：全流程托管式服务，解决客户"跨境对接"的所有麻烦</h5><p>核心逻辑：内地客户对接香港保险的核心痛点之一是"流程繁琐、两地衔接不便"，人设重点突出"省心"，打造"从需求分析到售后跟进"的全流程服务，让客户无需操心合规细节、对接细节，轻松完成跨境配置。</p><p>具体落地行为：</p><ul><li>前期：主动对接客户，梳理需求（无需客户主动了解港险细节），结合合规要求，筛选适配的持牌机构和产品，整理清晰的对比清单，节省客户筛选时间；</li><li>中期：协助客户完成前期准备（如赴港签证咨询、签约材料整理、香港持牌顾问对接），全程跟进转介流程，及时同步进度（如"已核验机构牌照""持牌顾问已联系客户""签约时间已确认"），避免客户与机构直接对接的繁琐；</li><li>后期：客户签约后，协助跟进冷静期事宜、保费到账确认，后续若有保单管理、理赔咨询、条款解读等需求，主动衔接香港持牌机构，为客户提供内地与香港的双向沟通桥梁，解决客户"售后找不到人"的顾虑；</li><li>增值服务：为客户提供跨境资产配置相关的辅助建议（如外汇兑换咨询、香港银行账户开通指引），贴合客户全方位的跨境需求，提升人设粘性。</li></ul><h4>二、能力筹备（四大核心能力，支撑人设落地）</h4><h5>（一）合规能力（基础必备，不可动摇）</h5><ul><li>核心知识：熟练掌握香港保监局《保险中介人管理守则》《转介费指引》（2025.10.1起实施），明确转介流程中的合规红线（如禁止内地签约、禁止返佣、转介费上限50%、持牌机构要求）；</li><li>实操能力：能独立通过香港保监局官网（<a href="https://iir.ia.org.hk" target="_blank">https://iir.ia.org.hk</a>）核验机构/个人牌照信息（包括牌照号码、有效期、业务范围），留存核验截图，规避违规合作风险；能准确解读转介协议核心条款，确保协议符合监管要求，避免条款漏洞；</li><li>风险把控：能快速识别违规行为（如机构声称"可内地签约""转介费超50%""私下返佣"），坚决拒绝合作，同时能向客户普及合规知识，引导客户选择合规渠道。</li></ul><h5>（二）跨境资产配置与港险专业能力（核心竞争力）</h5><ul><li>港险产品知识：深入了解香港主流合规保险产品（重疾险、储蓄分红险、年金险、医疗险）的特点、保障范围、理赔规则、适合人群，能根据客户需求精准匹配产品，不夸大产品收益；</li><li>跨境资产配置知识：了解内地客户跨境资产配置的常见需求（保值增值、分散风险、子女教育、养老规划），掌握跨境资产配置的基础逻辑，能结合香港保险产品，为客户提供简单、可落地的配置建议；</li><li>差异认知：清晰掌握香港保险与内地保险的核心差异（法律适用、理赔流程、货币风险、监管要求），能客观对比，帮助客户理性选择，避免客户因认知偏差产生误解。</li></ul><h5>（三）客户服务能力（人设落地关键）</h5><ul><li>需求挖掘能力：善于倾听客户需求，能从客户的年龄、职业、家庭结构、资产状况、需求痛点（如"想给孩子存海外教育金""担心养老保障不足"）出发，精准梳理核心需求，避免盲目推荐；</li><li>沟通表达能力：能用通俗的语言拆解专业术语（如"冷静期""首年佣金""美元保单汇率风险"），避免使用晦涩的行业话术，让内地客户轻松理解；同时能高效对接香港持牌机构，做好双向沟通，解决语言、地域带来的沟通障碍；</li><li>全流程跟进能力：建立客户服务台账，从需求对接、机构筛选、转介跟进，到售后咨询，全程记录，及时同步进度，让客户清晰了解每一步流程，提升客户信任感；</li><li>应急处理能力：面对客户疑问（如"理赔遇到问题""保单信息有误"），能快速响应，衔接香港持牌机构，协助解决问题，不推诿、不拖延。</li></ul><h5>（四）资源整合能力（提升人设竞争力）</h5><ul><li>合规资源整合：筛选并维护3-5家优质香港保监局持牌经纪公司（如康宏理财、赛诺保险经纪等），建立稳定的合作关系，确保为客户提供多样化的合规选择；</li><li>辅助资源整合：对接香港银行、签证咨询机构等，为客户提供外汇兑换、银行账户开通、赴港签约签证咨询等增值服务，打造"一站式"跨境配置服务体验；</li><li>客户资源维护：建立客户档案，定期跟进客户需求（如保单周年提醒、资产配置调整建议），提升客户粘性，通过老客户转介绍扩大影响力，强化"中立、专业、省心"的人设标签。</li></ul><h4>三、人设对外呈现（让客户快速感知核心特质）</h4><h5>（一）对外宣传核心话术（简洁好记，突出优势）</h5><ul><li>核心口号：跨境资产配置顾问｜港险合规转介专员，中立不推销，专业更省心，让内地客户对接港险更合规、更简单；</li><li>核心宣传语1：不绑定任何机构，只做你专属的跨境资产配置伙伴，合规为先，需求为王；</li><li>核心宣传语2：精通香港保监局合规规则，筛选优质持牌机构，全流程托管，让你赴港配置港险，省心、放心、合规；</li><li>常见疑问回应话术："我不推销任何一款产品，只根据你的需求，筛选合规、适配的香港持牌机构和产品""所有转介流程严格遵循香港保监局规定，转介费不超过首年佣金50%，全程透明可查""你无需操心对接细节，从需求分析到售后跟进，我全程帮你搞定"。</li></ul><h5>（二）形象与场景呈现（贴合人设，增强信任）</h5><ul><li>形象定位：专业、干练、亲和，避免过度商业化，突出"顾问感"，对外呈现时可搭配港险合规知识、跨境资产配置干货分享，强化专业标签；</li><li>场景呈现：日常可分享香港保监局政策解读、港险合规小知识、客户需求案例（隐去隐私）、跨境资产配置干货，让客户直观感受到"专业"；同时可展示合规合作机构的牌照核验截图、转介协议核心条款（隐去敏感信息），强化"合规"标签；</li><li>沟通场景：与客户沟通时，先倾听需求，再给出建议，不急于推销，主动普及合规知识，让客户感受到"中立"；全程耐心解答疑问，及时跟进进度，让客户感受到"省心"。</li></ul><h4>四、人设落地保障（长期坚持，强化标签）</h4><ul><li>定期自查：每季度梳理自身服务流程，检查是否符合香港保监局合规要求，是否坚守"中立、专业、省心"的核心特质，及时调整不足；</li><li>知识更新：每月学习香港保监局最新政策、港险产品迭代信息、跨境资产配置趋势，提升专业能力，确保给出的建议具备时效性；</li><li>客户反馈：定期收集客户反馈，了解客户对服务的满意度，针对客户提出的问题，优化服务流程，强化人设标签；</li><li>合规底线：始终坚守"仅对接香港保监局持牌机构、转介费不超50%、不参与受规管活动"的核心前提，不触碰违规红线，守护人设信任度。</li></ul>
