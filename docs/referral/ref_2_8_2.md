# 8.2 精细化结算流程

<h3>8.2 精细化结算流程</h3><p>转介费精细化结算流程严格遵循"合规优先、流程闭环、明细可查"原则，按"核算准备→明细确认→合规结算→闭环完成"四步有序推进，同时严格执行2025年10月香港保监局新规——转介费上限为首年佣金的50%。</p><h4>第一步：核算准备（结算前置，夯实基础）</h4><ul><li><strong>资格复核：</strong>再次复核保单已过21天法定冷静期、客户未退保、保单正常承保生效，三者同时满足方可启动核算。</li><li><strong>资料收集：</strong>收集保单生效证明、核保结果通知书、客户缴费凭证、转介人信息登记表、转介服务记录等。</li><li><strong>核心数据核对：</strong>核对保单数据（保单号、首年保费、首年佣金）、转介人收款账户信息；根据2025年10月新规，核算转介费≤首年佣金×50%。</li><li><strong>资料归档：</strong>建立专属结算档案，确保后续核查可追溯。</li></ul><h4>第二步：明细确认（双向核对，杜绝纠纷）</h4><ul><li><strong>生成结算明细：</strong>结算专员生成《转介费结算明细单》，明确标注保单号、客户姓名、首年佣金金额、转介费核算金额、支付方式、预计到账时间等。</li><li><strong>内部核对：</strong>合规部门复核转介费是否符合50%上限、数据是否与保单信息一致。</li><li><strong>转介人确认：</strong>发送明细单给转介人核对（通常3个工作日），转介人确认无异议后签署确认意见；若有异议，在核对期限内提出。</li></ul><h4>第三步：合规结算（按规支付，全程可控）</h4><ul><li><strong>结算申请：</strong>提交转介费结算申请，附上所有结算资料及确认文件，明确标注"符合2025年10月新规，转介费未超首年佣金50%"。</li><li><strong>支付操作：</strong>财务部门按约定支付方式（银行转账）将转介费足额支付至转介人收款账户，支付时备注"XX保单转介费"。</li><li><strong>新规合规把控：</strong>支付前再次核对转介费金额，未突破50%上限；若需突破上限，需提供香港保监局认可的说明材料及合规审批文件。</li></ul><h4>第四步：闭环完成（归档复盘，流程收尾）</h4><ul><li><strong>到账确认：</strong>支付完成后1-2个工作日内联系转介人确认到账，签署《转介费到账确认书》。</li><li><strong>资料全面归档：</strong>将所有结算资料（结算申请、明细单、审批文件、转账凭证、到账确认书等）统一整理归档。</li><li><strong>流程复盘：</strong>复盘结算流程，梳理是否存在资料遗漏、数据错误、时效延误等问题。</li><li><strong>进度同步：</strong>将结算结果同步给香港持牌顾问，完成系统录入，各方信息同步。</li></ul><h4>新规补充说明（2025年10月生效）</h4><ul><li>适用范围：所有香港保险转介业务，涵盖内地访客赴港投保相关转介服务，无例外情况。</li><li>特殊例外：仅当转介合作方为香港金管局、证监会、强制性公积金计划管理局所监管的公司时，50%上限要求不适用。</li><li>违规后果：若转介费结算超出首年佣金50%且无合规审批，将触发香港保监局监管处罚，需限期整改并追回超额部分。</li></ul>
