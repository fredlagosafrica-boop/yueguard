# 7.3 转介人维护动作

<h3>7.3 转介人维护动作</h3><p>核保及冷静期阶段，转介人核心定位为"进度同步者、基础服务者"，严格坚守角色边界，聚焦"进度跟进、疑问解答、权益提醒"三大核心动作。</p><h4>一、进度跟进（及时同步，让客户清晰掌握流程）</h4><ul><li><strong>核保阶段跟进：</strong>主动对接香港持牌顾问，每3个工作日询问一次核保进度，同步当前状态，确认进度后第一时间以文字形式同步给客户，不添加任何个人解读、不承诺核保结果。</li><li><strong>冷静期阶段跟进：</strong>保单出具后立即同步"冷静期已启动"；冷静期中期（第10-12天）提醒剩余时长；冷静期届满前1-2天同步"冷静期即将届满"。</li><li><strong>特殊情况跟进：</strong>若核保周期延长、需补充体检，或客户提出退保申请，第一时间同步客户，协助持牌顾问传递信息。</li></ul><h4>二、疑问解答（精准回应，不越界、不误导）</h4><ul><li><strong>可解答范围：</strong>核保、冷静期相关的基础流程类、时效类疑问（"核保还要多久""冷静期从哪天开始算""退保需要准备什么资料"等），不涉及产品条款、核保评估等专业内容。</li><li><strong>不可解答范围：</strong>核保结果原因、保单条款解读、退保损失（冷静期外）等专业问题，若客户提出此类疑问，需第一时间引导客户对接香港持牌顾问。</li><li><strong>解答规范：</strong>回应疑问时简洁、准确，不添加个人观点、不做额外承诺，若自身不确定答案，先对接持牌顾问确认后，再同步给客户。</li></ul><h4>三、权益提醒（前置告知，保障客户法定权益）</h4><ul><li><strong>冷静期权益提醒：</strong>明确告知"香港法定冷静期为21个自然日，从收到保单当日起算，冷静期内可无损失退保，仅扣除少量工本费"，提醒内容客观中立。</li><li><strong>核保相关权益提醒：</strong>体检/除外责任/加费/拒保等核保结果对应的权益提醒。</li><li><strong>禁忌提醒：</strong>不诱导客户退保、不劝阻客户退保，不强迫客户接受核保结果，尊重客户自主决策。</li></ul><h4>四、维护动作核心纪律</h4><ul><li>全程坚守角色边界，所有维护动作均围绕"进度、疑问、权益"展开，不触碰专业类、销售类操作。</li><li>所有沟通、提醒、跟进动作均留存记录，确保可追溯。</li><li>保持客观中立，不偏袒任何一方，不传递违规信息、不误导客户。</li><li>遇到自身无法处理的问题，第一时间对接香港持牌顾问，不擅自处理、不敷衍客户。</li></ul>
