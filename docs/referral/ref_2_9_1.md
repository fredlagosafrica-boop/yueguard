# 9.1 精细化售后维护

<h3>9.1 精细化售后维护</h3><p>保单过21天法定冷静期、正式生效后，进入长期售后维护阶段。本阶段核心原则为"专业事宜归专业、日常维护重价值"，转介人严格坚守角色边界，不介入任何专业类操作，聚焦"桥梁衔接+价值传递"两大核心。</p><h4>一、专业售后：坚守边界，做好"桥梁传话"</h4><p><strong>核心定位：</strong>转介人在专业售后环节仅承担"桥梁传话"角色，所有保单相关专业事宜均由香港持牌团队全程负责。</p><ul><li><strong>专业事宜界定：</strong>保单保全（投保人信息变更、受益人变更、缴费方式调整、保单贷款等）、理赔咨询及办理、保单条款解读、收益查询及解读、保单升级/变更相关专业建议等。</li><li><strong>"桥梁传话"操作规范：</strong>客户发起专业咨询时，第一时间告知客户"相关专业事宜将由香港持牌团队为您专业解答"，立即将客户疑问完整传递给香港持牌顾问，不添加个人观点、不遗漏诉求。持牌团队给出答复后及时同步给客户，全程不解读、不修改专业意见。</li><li><strong>禁止动作：</strong>严禁擅自解答保单专业问题、代办保单保全/理赔、解读保单条款及收益、给出任何与保单相关的专业建议。</li></ul><h4>二、日常维护：传递价值，做好精细化陪伴</h4><p><strong>内容定位：</strong>聚焦跨境理财、汇率相关的常识性、资讯类内容，内容需合规、客观、实用，不涉及产品推荐、收益承诺。</p><ul><li><strong>发送内容：</strong>跨境理财常识（香港理财市场基础常识、跨境资金往来合规要点、家庭资产配置基础理念）、汇率资讯（港币与人民币汇率走势、汇率波动应对小技巧）、其他合规价值内容（香港保险行业合规动态、客户权益保护常识）。</li><li><strong>发送规范：</strong>频率建议每月1-2次；形式以文字、图文结合为主，内容简洁易懂；所有发送内容需经过合规审核，确保无违规表述、无虚假信息、无产品推荐。</li><li><strong>互动维护：</strong>发送内容后简单询问客户反馈，若客户提出专业类疑问，及时引导对接香港持牌团队。</li></ul><h4>三、售后维护核心纪律</h4><ul><li>全程坚守角色边界，专业事宜全部交由香港持牌团队负责，转介人仅做"桥梁传话"。</li><li>日常维护内容需合规、客观，不涉及产品销售、专业解读、收益承诺。</li><li>所有售后沟通、内容发送记录均需留存，确保可追溯。</li><li>尊重客户意愿，若客户明确表示无需接收日常资讯，及时停止发送。</li><li>不干预客户的保单决策，不诱导客户办理保单保全、理赔、退保等操作。</li></ul>
