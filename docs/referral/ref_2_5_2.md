# 5.2 转介人精细化辅助动作

<h3>5.2 转介人精细化辅助动作</h3><p>转介人在本环节核心定位为「合规辅助者、流程衔接者」，严格坚守4.2章节角色边界，不介入任何产品讲解、方案设计、条款解读等销售类动作，仅聚焦赴港辅助、证件提醒、对接协调三大核心场景，提供精细化、高效化衔接服务，助力客户顺利完成赴港签约，同时配合持牌顾问推进服务流程，提升客户体验，全程不越位、不缺位，贴合港险全流程合规要求。</p><h4>一、赴港辅助：全流程行程支持</h4><p>核心聚焦客户赴港行程相关的基础服务，不涉及任何港险专业内容。</p><ul><li><strong>基础疑问精准解答：</strong>回应客户关于赴港的基础问题（通行证办理、出入境规定、交通路线、住宿推荐等），对于产品、条款、收益等专业疑问，第一时间引导其对接香港持牌顾问。</li><li><strong>行程规划协助衔接：</strong>建议客户提前1天抵达香港，结合签约网点位置推荐便捷交通和住宿，提醒预留充足签约办理时间，避开出入境和交通高峰。</li><li><strong>突发情况居中协调：</strong>若出现行程延误、路线迷路、证件临时遗失等突发情况，第一时间衔接持牌顾问，协助双方沟通调整签约安排，全程仅负责居中传递信息。</li></ul><h4>二、证件提醒：前置提醒、逐项核对</h4><ul><li><strong>首次前置提醒（赴港前7-10天）：</strong>主动对接持牌顾问获取「签单证件清单」，以文字形式同步给客户，明确标注每种证件要求。</li><li><strong>分身份细化提醒：</strong>内地普通居民（身份证、通行证、银行卡、住址证明）；企业主（额外营业执照复印件、个人资产证明）；自由职业者（额外收入证明）。</li><li><strong>二次核对确认（赴港前1-2天）：</strong>再次私信客户逐项核对所有证件，重点核对通行证签注有效期、证件原件是否齐全。</li><li><strong>赴港当天最终提醒：</strong>出发前再次发送私信确认，确保所有证件资料已携带。</li></ul><h4>三、对接协调：高效衔接</h4><ul><li><strong>沟通时间精准协调：</strong>提前询问客户空闲时间，同步给持牌顾问，协助约定合适沟通时段。</li><li><strong>沟通时间双向提醒：</strong>提前1-2小时分别提醒客户和持牌顾问，若任何一方无法按时沟通，第一时间同步给另一方协助协调。</li><li><strong>沟通卡点及时化解：</strong>若沟通中出现节奏不一致，主动居中提醒，协助传递基础信息，不介入沟通内容。</li><li><strong>流程进度定期同步：</strong>定期（建议每3天）在三方群内同步当前服务进度。</li></ul><h4>核心纪律要求</h4><ul><li>严格坚守角色边界，所有辅助动作围绕赴港行程、证件准备、沟通协调展开，绝不触碰产品讲解、方案设计、条款解读等销售类动作。</li><li>所有辅助沟通、提醒、协调动作均留存记录，配合持牌顾问完成全流程合规留痕。</li><li>遇到自身无法解答的问题，第一时间引导客户对接香港持牌顾问，不随意答复。</li><li>全程保持中立、客观，专注做好衔接服务。</li></ul>
