# 9.3 裂变辅助动作

<h3>9.3 裂变辅助动作</h3><p>裂变辅助动作核心是"合规引导、价值赋能"，转介人需以优质服务为基础，通过适当的增值服务增强老客户粘性、激发老客户转介绍意愿，严禁任何形式的"返佣裂变"，坚守合规底线。</p><h4>一、核心辅助动作：提供合规增值服务</h4><p><strong>增值服务范围（合规、非利益性）：</strong></p><ul><li><strong>资讯类：</strong>定期为老客户推送专属跨境理财干货、汇率深度解读、香港保险行业合规动态。</li><li><strong>服务类：</strong>为老客户提供保单基础管理提醒（缴费日期提醒、保单状态同步）、跨境出行基础协助、售后衔接优先服务。</li><li><strong>知识类：</strong>定期组织合规的跨境理财、香港保险基础常识分享（线上简短分享、图文手册推送）。</li><li><strong>情感类：</strong>节日问候、客户重要节点（如生日）祝福，拉近与客户的距离。</li></ul><p><strong>增值服务执行规范：</strong></p><ul><li>频率适度：每月1-2次，避免过于频繁打扰客户。</li><li>合规可控：所有增值服务均不涉及现金、返佣、实物奖励等利益性激励，不与转介绍数量挂钩。</li><li>自愿原则：增值服务由老客户自愿选择参与，不作为转介绍的前提条件。</li></ul><h4>二、绝对禁止：严禁"返佣裂变"，坚守合规底线</h4><p><strong>严禁返佣行为界定：</strong></p><ul><li>现金返佣：以任何形式向老客户支付现金、转账，作为转介绍奖励；</li><li>实物返佣：向老客户赠送贵重实物（家电、首饰、购物卡等）与转介绍挂钩；</li><li>利益置换：以减免费用、提供利益性服务等作为转介绍激励；</li><li>间接返佣：通过第三方、关联方向老客户提供利益激励。</li></ul><p><strong>违规后果：</strong>触发香港保监局监管处罚，被纳入行业黑名单，面临罚款、承担法律责任；转介费不予结算，已结算的需全额追回。</p><h4>三、裂变辅助动作核心要求</h4><ul><li>所有裂变辅助动作均以"合规"为前提，不触碰返佣红线，坚守角色边界。</li><li>增值服务聚焦"价值传递、客户陪伴"，不与转介绍数量、转介绍成果挂钩。</li><li>转介绍引导坚持"自愿、温和"原则，不强迫、不诱导，尊重老客户意愿。</li><li>所有裂变辅助动作、沟通记录均需完整留存，确保可追溯。</li><li>若发现违规苗头（如老客户提出返佣要求），需明确拒绝，做好合规解释。</li></ul>
