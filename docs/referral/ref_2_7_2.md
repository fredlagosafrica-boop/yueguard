# 7.2 冷静期相关规定

<h3>7.2 冷静期相关规定</h3><p>香港保险冷静期为香港保险监管局明确规定的法定权益，旨在保障客户投保知情权，给予客户充分的时间复核保单条款、确认投保意愿。</p><h4>一、冷静期核心界定</h4><ul><li><strong>法定时长：</strong>香港保险法定冷静期为21个自然日，该时长为监管硬性规定，任何保险公司、任何险种均需严格执行，不得缩短、延长或豁免。</li><li><strong>起算时间：</strong>冷静期从客户实际收到正式保单（纸质保单或电子保单）之日起计算，精确到自然日，而非保单生效日、签单日或核保通过日。</li><li><strong>生效前提：</strong>冷静期仅在保单正式出具、客户成功收到保单后启动，未收到正式保单前，不进入冷静期。</li></ul><h4>二、冷静期内核心权益（无损失退保）</h4><ul><li><strong>退保权限：</strong>客户可无任何理由、无任何限制，向香港保险公司提出退保申请，保险公司不得拒绝，不得收取任何违约金、手续费（不含少量保单工本费）。</li><li><strong>保费退还：</strong>客户申请退保后，保险公司需在规定时效内（通常3-5个工作日），将客户已缴纳的首期保费全额退还（扣除少量工本费）。</li><li><strong>退保流程：</strong>客户需向持牌顾问提交书面退保申请（按保险公司标准格式填写），并提供相关凭证。</li></ul><h4>三、冷静期内注意事项</h4><ul><li>冷静期内退保无任何损失，不影响后续再次投保香港保险，无不良记录。</li><li>冷静期届满后，客户再提出退保，将按保单现金价值计算退保金额，可能产生较大资金损失。</li><li>持牌顾问需在客户收到保单当日，再次提醒客户冷静期时长、起算时间及无损失退保权益。</li><li>转介人不得在冷静期内诱导客户退保、劝阻客户退保，不得干预客户退保决策。</li><li>若客户在冷静期内提出退保，保险公司将终止保单效力，转介费结算条件不成立，不予结算转介费。</li></ul><h4>四、特殊情况说明</h4><p>若客户因特殊原因（如未及时收到保单、保单信息错误）导致冷静期计算有误，需及时联系香港持牌顾问，由持牌顾问向保险公司申请核实、调整，调整后的冷静期起算时间以保险公司官方确认为准。</p>
