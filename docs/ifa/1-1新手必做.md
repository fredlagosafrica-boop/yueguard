# 1.1 新手必做

> 入职前必做的准备工作

## 核心任务清单

- [ ] 完成保险中介人资格考试报名
- [ ] 准备入职资料
- [ ] 开通香港银行账户
- [ ] 完成公司入职流程

## 资格考试准备

香港保险业监管局（IA）规定，所有保险中介人必须通过资格考试。

**考试科目：**
- 香港保险原理及实务
- 一般保险
- 人寿保险

**报名方式：** 通过香港保险业监管局网站或指定考试中心报名。

## 入职资料准备

| 资料 | 说明 | 状态 |
|------|------|------|
| 身份证 | 有效身份证明文件 | ⬜ |
| 港澳通行证 | 有效期内 | ⬜ |
| 学历证明 | 毕业证、学位证 | ⬜ |
| 无犯罪记录证明 | 三个月内有效 | ⬜ |
| 照片 | 证件照 | ⬜ |

## 银行账户开通

建议选择：**中银香港** / **汇丰银行** / **渣打银行**

## 入职流程

```
提交资料 → 面试 → 培训 → 考试 → 持牌 → 入职
```

> 注意：整个流程通常需要2-4周，请提前规划时间。
