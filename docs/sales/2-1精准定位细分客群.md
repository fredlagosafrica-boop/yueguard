# 2.1 精准定位细分客群

> 找准目标客户，精准营销

## 客群分类概览

### 按需求分类

1. **财富传承型** - 高净值客户，关注资产传承
2. **跨境家庭保障型** - 有内地-香港跨境生活需求
3. **留学移民规划型** - 子女教育或移民需求
4. **中产阶级储蓄增值型** - 稳健增值需求
5. **风险需求专项型** - 特定风险保障需求

### 按人生阶段分类

- 💼 **职场新人** - 30岁前
- 👨‍👩‍👧 **家庭形成期** - 30-40岁
- 🏠 **家庭稳定期** - 40-50岁
- 👴 **退休规划期** - 50岁后

## 选择客群的核心原则

> 🎯 选择比努力更重要。找准目标客群，精准切入，比泛泛而谈有效10倍。

### 判断标准

✅ 有明确需求（子女教育、养老、资产传承）
✅ 有支付能力（年收入符合产品要求）
✅ 有决策权（本人或直系亲属）
✅ 有合理期望（了解保险产品特性）
