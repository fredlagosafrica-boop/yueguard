# 赴港投保硬性合规要求（缺一不可）

<h3>赴港投保硬性合规要求（缺一不可）</h3><p><strong>身份核验：</strong>投保人必须亲自赴港，本人亲临香港完成：身份核验、见证签署、录音录像、风险告知</p><p><strong>禁止：</strong>代签、远程签、内地异地签、视频代替面签、代办投保</p><p><strong>证件资质：</strong>合法有效证件（内地身份证、港澳通行证/护照有效签注）；地址证明合规（水电煤账单、银行账单等），禁止伪造</p><p><strong>流程地点：</strong>必须在香港本地合规营业网点/保险公司授权签约中心办理；禁止私人住宅、酒店客房、内地会议室私下签单</p>
