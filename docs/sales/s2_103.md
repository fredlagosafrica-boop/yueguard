# 七项核心内容（缺一不可）

<h3>七项核心内容（缺一不可）</h3><p><strong>1.产品名称：</strong>明确产品全称（如香港XX终身寿险），不使用简称或模糊表述</p><p><strong>2.保费细节：</strong>明确年缴/月缴金额、总保费，标注货币类型（美金/港币），无隐藏费用</p><p><strong>3.保障范围/收益预期：</strong>保障型：覆盖疾病、医疗项目、理赔额度；收益型：预期收益区间，不夸大、不隐瞒</p><p><strong>4.缴费年限：</strong>明确缴费期（如10年缴、20年缴），标注是否支持灵活缴费</p><p><strong>5.合规投保流程：</strong>列出投保步骤，明确责任方和时间节点</p><p><strong>6.风险提示：</strong>明确核心风险（分红险：预期分红≠保证分红；重疾险：除外责任、退保损失）</p><p><strong>7.理赔路径：</strong>明确理赔申请材料、提交渠道、审核周期、赔付到账方式</p>
